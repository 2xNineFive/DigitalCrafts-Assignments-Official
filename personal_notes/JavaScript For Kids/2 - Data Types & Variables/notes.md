## Incrementing and Decrementing 
**Incrementing**
means increasing by 1

```javascript
++someVariable;
```

**Decrementing**
means decreasing by 1

```javascript
--someVariable;
```

**Example**
```javascript
var dope = 0;
++dope;
// 1
++dope;
// 2
++dope;
// 3
--dope;
// 2
--dope;
// 1
```


### `+=` & `-=`
Both of these are operators that can be used to make incrementing and decrementing variables easier. 

**Ex. Hard Way**
```javascript
var dope = 10 
dope = dope + 1; 
dope;
// 11
```


**Ex. Easier Way**
```javascript
var doper = 10;
doper += 2;
// 12
doper -= 10;
// 2
```


### `*=` & `/=`
Both of these are operators that can be used to make *multiplying* and *dividing* variables easier. 

**Example**
```javascript
var doper = 10;
doper *= 2;
// 20
doper /= 5;
// 10
```
